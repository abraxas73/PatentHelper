### 1. 매핑 추출 로직 개선

#### 1.1 복수 매핑 처리 문제 해결
- **문제**: "120: GMT 판재 121: 앵글"처럼 한 줄에 여러 매핑이 있을 때 첫 번째만 추출
- **해결**: 정규식 패턴을 non-greedy(`*?`)로 변경하고 lookahead assertion 추가
```python
# 수정 전
'basic': r'(\d{1,4}[a-zA-Z]?)\s*[:：]\s*([가-힣\w\s]+)'

# 수정 후
'basic': r'(\d{1,4}[a-zA-Z]?)\s*[:：]\s*([가-힣][가-힣\w\s]*?)(?=\s*\d{1,4}[a-zA-Z]?\s*[:：]|$|\n)'
```

#### 1.2 관형어구 처리 개선
- **추가된 관형어구 패턴**:
  - "~를 위한", "~하기 위한", "~시키기 위한"
  - "~를 통해", "~를 통한", "~를 이용한"
- **개선사항**:
  - 관형어 제거 후 2글자 미만이 되면 원본 유지
  - "회전하는 축" → "회전하는 축" (1글자 "축"으로 축소되는 것 방지)

#### 1.3 부사구 제거 로직 추가
- **문제**: "각각 체결됨과 동시에 컵와셔(86)"에서 전체가 추출됨
- **해결**: 부사구 제거 패턴 추가
  - "동시에", "즉시", "함께", "순차적으로", "동일하게", "각각" 등
  - 문장 시작과 중간 모두 처리

#### 1.4 공백 개수 제한
- 정리된 라벨에서 공백이 2개를 초과하면 해당 매핑 제외
- 너무 긴 문장이 매핑으로 추출되는 것 방지

### 2. 어노테이션 위치 문제 수정

#### 문제
- 모든 어노테이션이 이미지 왼쪽에 붙어서 표시됨

#### 원인
- 확장된 캔버스의 중심을 기준으로 왼쪽/오른쪽 판단

#### 해결
```python
# 수정 전
center_x = region['center']['x'] + left_expansion
if center_x < (expanded_width / 2):  # 확장된 캔버스 기준

# 수정 후
original_center_x = region['center']['x']
if original_center_x < (original_width / 2):  # 원본 이미지 기준
```

### 3. 프론트엔드 개선

#### 3.1 이미지 모달 NaN 표시 문제 해결
- **문제**: "페이지 NaN • × px" 표시
- **원인**: width/height가 'Unknown' 문자열이거나 undefined
- **해결**:
  - width/height가 유효한 숫자일 때만 크기 정보 표시
  - original_page가 undefined일 때 대체 텍스트 표시

#### 3.2 이미지 다운로드 기능 개선
- **문제**: AWS S3/CloudFront URL에서 다운로드 대신 새 탭으로 열림
- **원인**: CORS 정책으로 인한 download 속성 미작동
- **해결**:
  - fetch API로 이미지를 blob으로 변환
  - blob URL 생성 후 다운로드
  - 에러 발생 시 새 탭으로 열기 (fallback)
